```
バブルエディタの仕様
バブルエディタは、リンクを生成したときに、そのリンク先がない場合に、新しい項目を生成する。
新しい項目は、エディタとなって追加される。
この新しい項目は、結果として、記事となる。

たとえば「ポワント」という言葉をリンクした時、
---
[ポワント](ポワント)の詳しい解説。
---
この時には、新しいエディタが生成され、
---
# ポワント
---
と書かれる。
内部的には、バブルチェッカーが起動して、新しい項目を立ち上げるのかを管理する。
これにより、全ての内部リンクが関連付けられ、強力な内部リンクをもった、サイトが完成する。
１００から１０００ぐらいの用語があるサイトで効果を発揮するはずである。
```
